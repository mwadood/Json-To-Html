Table

REQUIRED PARAMETER(s)
  1. Data: Jason object
    NOTE: BY DEFAULT TABLE APPEND TO HTML BODY ELEMENT See Example
OPTIONAL PARAMETER(s)
  1. TableID: Table id if you want to assign specific id to table
    Data Type: string
  2. AppendTo: HTML Element such as 'div' ID to append table to, by default table will be append to 'body'.
    Data Type: string See Example
  3. DefaultHeader: By default 'True' show the default header.
    Data Type: Boolen
  4. CustomerHeader: Set column(s) to display specific columns.
    Data Type: JOSN Object See Example
  5. AddToColumn: Append or Prepend value to column.
    Data Type: JOSN Object See Example Default Columns See Example Custom Columns
  6. Sort: Boolen value by default 'True' show the default header.
    Data Type: Boolen
  7. UpdateFunction: To update the table row.
    Data Type: Function See Example
  8. CreateFunction: To create new table's row.
    Data Type: Function See Example
  9. DeleteFunction: To create new table's row.
    Data Type: Function See Example
